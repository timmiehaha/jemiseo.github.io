projects

Here are some of my projects:

flavordex
A tiny web app I wrote with the Holy Trinity of Web Development, which runs with Node.js, uses Express for routing, and MongoDB for its database. It is basically a collection of item flavor texts from the Pok√©mon games. My motivation behind creating this was that it would help expose me to more Korean vocabulary.
tweetdelete
A tool I wrote with Python using the Tweepy API. It allows the user to go through their Twitter account and manage (primarily delete) even their oldest Tweets that may not show up on their account due to Twitter's limitations. *The app itself is a command line script but having an image of the Terminal is kinda boring. I hope this isn't false advertising...
korean-lookup
A Java GUI that I wrote a while back when I didn't know how to use a native Korean keyboard. Works by converting English phonemes to their Hangeul representations and implements a web-scraper to retrieve definitions from specific websites.
spla2notify
A Discord bot I wrote in Python and hosted with AWS. Allows Splatoon 2 players to "subscribe" and "unsubscribe" to their desired map/mode combinations. When said map/modes are in rotation, the bot automatically notifies subscribed players through Direct Message. *No longer being hosted.
